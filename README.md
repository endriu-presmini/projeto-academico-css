# 💼 Site de Serviços — Projeto Acadêmico

Site institucional desenvolvido como exercício acadêmico para praticar propriedades fundamentais do CSS, apresentando serviços de desenvolvimento web, aplicativos e design.

## 🚀 Tecnologias

- HTML5
- CSS3

## ✨ Propriedades praticadas

- Flexbox (display: flex, flex-direction, gap)
- Estilização de header e footer
- Border-radius
- Centralização de elementos
- Estrutura semântica com section e main

## 📁 Estrutura

```
├── index.html
└── style.css
```
